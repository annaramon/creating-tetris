# Tetris

A Python implementation of the classic Tetris game using object-oriented programming.

## Architecture

| File | Responsibility |
|------|---------------|
| `gameboard_class.py` | Game board logic — grid management, piece placement, collision detection, line clearing |
| `myplayer_class.py` | Player controls — piece movement, rotation, scoring |

## How to Run

```bash
python gameboard_class.py
```

## Tech Stack

- Python
- Object-Oriented Design
